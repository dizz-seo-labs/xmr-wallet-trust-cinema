# XMR Wallet Safety Field Manual

Static Astro SEO microsite for `https://xmrtrust.uk/`.

## Purpose

Independent educational Cake Wallet / Monero wallet-safety field manual focused on:

- seed phrase phishing prevention;
- Cake Wallet download-source checks;
- restore and restore-height safety;
- fake support and recovery-service red flags;
- non-custodial wallet responsibility;
- realistic Monero privacy limits.

## Safety boundaries

- Not an official Cake Wallet, Cake Labs, Monero.com, or Monero Project website.
- No financial, legal, privacy, security, recovery, or anonymity guarantees.
- Never collects or requests seed phrases, private keys, wallet files, screenshots, account identifiers, or support tickets.
- Sponsored outbound links must be disclosed and use `rel="sponsored nofollow noopener"`.
